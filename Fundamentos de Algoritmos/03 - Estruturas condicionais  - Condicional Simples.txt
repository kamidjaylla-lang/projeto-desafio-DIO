Condição: Estado de uma pessoa ou coisa.

Condicional: Expressa uma condição ou suposição, contem ou implica uma suposição ou hipótese.

Tipos de Estruturas Condicionais:

-Simples: A condição deve ser satisfeita de acordo com a operação.
-Composta: Pode haver exceções caso a condição não seja satisfeita.
-Encadeada: Condições e operações estão presentes dentro de outras formando uma "cadeia" sucessiva permitindo varias alternativas possíveis.

Operadores Relacionais:

Símbolo    Significado
   |            |
   =         Igual a.
   <>     Diferente de.
   >        Maior que.
   <        Menor que.
   >=     Maior ou igual a.
   <=     Menor ou igual a.

 Exemplo de Estrutura Condicional Simples:
 
           Inicio                    Inicio do programa:
             |                     Variáveis
            A,B                       Pontuação 1, Pontuação 2, Total: Inteiro  
             |                     Inicio 
         X <- A + B                   // 1. Entrada de dados
             |                        Ler Semestre 1                                
         N--X>10--S                   Ler Semestre 2
         |        |                   
         |        V                   // 2. Processamento
         |_ Fim  _|                   Total <- (Semestre 1 + Semestre 2)/2
                                        
                                      // 3. Condicional Simples
                                      SE Total >= 10 ENTÃO 
                                         ESCREVER "Aprovado! Total de Pontos:", Total
                                      FIM SE

A estrutura condicional simples é um comando de programação que executa um bloco de código somente se uma condição for verdadeira. Se a condição for falsa, o bloco é simplesmente ignorado e a execução do programa continua. A estrutura utiliza a palavra-chave SE (ou if em outras linguagens) para tomar essa decisão. 

- A estrutura condicional simples é um comando que permite ao programa tomar uma decisão.
- Ela avalia uma condição, que pode ser verdadeira ou falsa.
- Se a condição for verdadeira, um conjunto de instruções é executado.
- Se a condição for falsa, nada acontece e o programa prossegue para a próxima instrução após a estrutura. 

Modo de operação: se (<condição>) então
                      <instruções para condição verdadeira>
                  fim_se
